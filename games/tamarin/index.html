About the Game

Tamarin is a fast-paced, physics-based multiplayer game inspired by Gorilla Tag. Climb, leap, and swing through the trees using only your hands — no buttons, no joysticks, just pure motion.

🛠️ Features

  • ✋ VR-based hand movement
  • 🌐 Online multiplayer lobbies
  • 🧩 Minigames and tag modes
